# Mac Terminal Practice

MacのTerminal操作に不慣れなメンバー向けの、UNIX基本コマンド練習ページです。

GitHub Pagesでそのまま公開できる、単一HTMLの教材として作成しています。

## 公開ページ

https://akilab.github.io/mac-terminal-practice/

## 内容

- Terminalの安全な練習ルール
- ホームディレクトリ直下の `~/practice` を起点にした演習
- 基本コマンドの説明とよく使うオプション
- ブラウザ上で入力できる打鍵練習
- 正誤判定と進捗表示
- Apacheログを使ったパイプライン分析練習
- `grep` 専用の検索トレーニング
- `find` 専用のファイル探索トレーニング
- `wget` / `curl` 専用のHTTP取得トレーニング
- 環境変数、シェル設定、PATHの基礎トレーニング
- Git / GitHubのブランチ、commit、push、Pull Request練習
- プロセス、ジョブ、ポート、ネットワーク確認、ログ監視の基礎トレーニング
- 権限、所有者、実行権限、`sudo` の基礎トレーニング

## 練習できる項目

`pwd`, `ls`, `cd`, `mkdir`, `touch`, `cat`, `cp`, `mv`, `rm`, `grep`, `head`, `tail`, `less`, `wc`, `sort`, `uniq`, `cut`, `find`, `wget`, `curl`, `env`, `export`, `alias`, `git`, `ps`, `pgrep`, `top`, `kill`, `jobs`, `fg`, `lsof`, `nc`, `chmod`, `sudo`, `whoami`, `id`, リダイレクト, パイプ

## ファイル構成

- `index.html`: 学習ガイドの入口ページ
- `mac-kitting-guide.html`: AI開発のためのMac初期設定ガイド
- `mac-terminal-practice.html`: UNIX基本コマンド練習ページ
- `mac-command-practice.html`: Apacheログを使ったコマンド組み合わせ練習ページ
- `mac-grep-practice.html`: `grep` 専用の検索練習ページ
- `mac-find-practice.html`: `find` 専用のファイル探索練習ページ
- `mac-curl_wget-practice.html`: `wget` / `curl` 専用のHTTP取得練習ページ
- `mac-env-path-practice.html`: 環境変数、シェル設定、PATH練習ページ
- `mac-git-practice.html`: Git / GitHub練習ページ
- `mac-process-job-practice.html`: プロセス、ジョブ、ポート、ネットワーク確認練習ページ
- `mac-permission-practice.html`: 権限、所有者、実行権限練習ページ

## 使い方

1. GitHub Pagesでこのリポジトリを公開します。
2. ブラウザでページを開きます。
3. `~/practice` を作成し、ページの説明に沿ってコマンドを練習します。

入力チェックはブラウザ内だけで動作し、入力内容は外部へ送信されません。
