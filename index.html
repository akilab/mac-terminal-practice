How To Use

使い方

入力チェックはブラウザ内だけで動作し、入力内容は外部へ送信されません。

1. ページを開く

この入口から、学習したい教材へ移動します。

2. Terminalで試す

教材内の準備コマンドを実行し、実際のファイルで練習します。

3. ブラウザで打鍵する

演習欄にコマンドを入力し、形を体に入れていきます。

Mac Terminal Practice / GitHub Pages compatible single HTML guide.
